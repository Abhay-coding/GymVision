## рҹҺҜ Overview

**AI Gym Coach** is a browser-based personal trainer powered by computer vision and AI. Point your webcam at yourself, pick an exercise, and it maps your skeleton across 33 keypoints in real time, counts every rep automatically, and speaks personalized form corrections through your speakers вҖ” no app, no subscription, no trainer fee.

## рҹҡҖ Quick Start

### Python

```bash
git clone https://github.com/your-username/GymVision.git
cd GymVision/MainPage

python -m venv venv && source venv/bin/activate
pip install -r requirements.txt

echo "ANTHROPIC_API_KEY=sk-ant-..." > .env
streamlit run main.py
```

### Docker рҹҗі

```bash
cd GymVision/MainPage
docker build -t gymvision .
docker run -p 8501:8501 -e ANTHROPIC_API_KEY=your_key gymvision
# вҶ’ http://localhost:8501
```
